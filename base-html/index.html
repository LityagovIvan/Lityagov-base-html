HTML-документ - это обычный текстовый документ может быть создан как в обычном текстовом редакторе(Блокнот), так и в специализированном, с подстветкой кода (Notepad++, Visual Studio Code и т.п.). имеет расширение .html.
HTML-документ состоит из дерева HTML-элементов и текста. Каждый элемент обозначается в исходном документе начальным (открывающим) и конечным (закрывающим) тегом (за редким исключением).
Начальный тег показывает, где начинается элемент, конечный где заканчивается. Закрывающий тег путем добавления слэша / перед именем тега: ‹имя тега›...‹имя тега›. Между начальным и закрывающим тегом находится содержимое элемента - контент.

Например:

  1. ‹p› Контент‹/p› - Создаёт параграф
  2. ‹b› Контент‹/b› - Делает полужирное начертание
  3. ‹i› Контент‹/i› - Курсивное начертание
